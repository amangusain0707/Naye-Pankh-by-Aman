# NayePankh Foundation Landing Page

## Overview

This project was developed as part of the Front-End Development Internship Selection Process conducted by NayePankh Foundation.

The objective was to create a modern and visually appealing webpage that represents the mission, impact, and initiatives of NayePankh Foundation while demonstrating front-end development skills.

The website highlights the foundation's work in food distribution, education support, sanitary pad distribution, community outreach, and volunteer engagement.

---

## Features

* Modern responsive user interface
* Interactive hero section
* Custom particle background using Three.js
* Smooth animations using GSAP
* Dark-themed design with neon visual effects
* Impact statistics section
* About Us section
* Internship Program section
* Donation and volunteering section
* Contact and social media links
* Mobile-friendly layout

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP Animation Library
* Three.js
